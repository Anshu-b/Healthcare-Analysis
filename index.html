This is a dynamic bar chart showing post-meal glucose changes (measured in mg/dl) for different meals, with negative values (red) indicating glucose decreases and positive values (green) indicating increases. Users can select different time intervals (30, 60, 90, or 120 minutes) using the buttons at the bottom to see how glucose levels change at various points after consuming these meals.



We chose to represent the data as a diverging green red scale heatmap utilizing a correlation matrix implemented with a helper function to compute the Pearson correlation coefficient r, as well as lay down the frameworks for the matrix to impute the data into. Color encoding similar to what you would see in clinical trials and experiments' results
